### 好买衣虚拟试衣间

好买衣虚拟试衣间是目前公司的核心项目，也是唯一的主打产品，目前已经与 100多家天猫一线女装品牌达成合作，服务了近 200万人次的用户。

* 负责 PC 版虚拟试衣间的研发及版本迭代。研发过程中，利用原生 Js 的可塑性，成功地在当时局限较多的 TAE 平台上运行完整的 SPA 项目，为之后的版本迭代奠定基础。在此期间，编写并开源了基于 Kissy 的类 jQuery 的工具库 [Mee.js](https://github.com/Amery2010/MeeJs)，这也使得在 TAE 平台中引入 jQuery 插件成为可能。

* 负责 TAE 平台转 JAE 平台的迁移重构工作。JAE 平台是淘宝当时推出的二代应用托管引擎，性能和自由度上有较大提升，因而使之前在 TAE 上无法实现的一些功能成为可能。在重构过程中，引入 Backbone，将 MVC 的概念带到当时项目中。重构过程中，利用自己实现的 [promise-lite.js](https://github.com/Amery2010/promise-lite)成功地解决了请求数据时经常遇到的 “callback hell” 问题。

* 参与 Mobile 版的虚拟试衣间的研发。与当时新进来的同事一起合作开发了手机端的试衣间版本。在原先的版本上通过多次系统性地优化，使应用的首屏加载速度从 2~3s降低到 1.2s，提高了首屏用户的留存率。之后，我推动并完成对手机端应用的重构工作，引入轻量级的 ratchet 框架并负责编写了底层的 MVC 架构，此次重构使得原先的项目依赖更少变得更加轻量级。

* 在公司与天猫新风尚的合作项目中对试衣间列表页进行深度优化，通过无限滚动以及 Ajax 数据缓存等方式，解决中低端机的加载效率，使应用在 80万的用户量下依然保持流畅。

* 参与好买衣虚拟试衣间的 ES6 重构工作。新版的虚拟试衣间基于 ES6，引入 Vue 2.0、VueX 进行改版。
